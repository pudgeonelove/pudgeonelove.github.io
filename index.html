Урок 1: Базовые настройки

Конфигурация (Переменные)

Зададим настройки сервера. Переменные помогают хранить важные данные в одном месте.

server.py 
name = "Alpha-Server"
port = 8080
print("Запуск на порту:", port)
> Запуск на порту: 8080

Математика нагрузки

Рассчитаем время передачи данных в системе.

calc.py 
size = 1024
speed = 10
print("Время:", size / speed, "сек.")
> Время: 102.4 сек.

Условия (If)

Проверка доступа к системе через IP.

auth.py 
user_ip = "127.0.0.1"
if user_ip == "127.0.0.1":
    print("Доступ разрешен")
> Доступ разрешен

Списки (Lists)

Очередь системных задач.

tasks.py 
queue = ["Backup", "Update", "Scan"]
print("Первая задача:", queue[0])
> Первая задача: Backup

Пожарная безопасность (Циклы)

Автоматический опрос датчиков дыма.

fire_safety.py 
sensors = ["Hall", "Kitchen", "Garage"]
for s in sensors:
    print("Проверка датчика:", s, "- OK")
> Проверка датчика: Hall - OK
> Проверка датчика: Kitchen - OK
> Проверка датчика: Garage - OK

Функции (Def)

Создание команды для пинга сервера.

ping.py 
def ping(ip):
    print("Пинг", ip, "завершен")

ping("8.8.8.8")
> Пинг 8.8.8.8 завершен

Словари (Dict)

Хранение данных администратора.

admin.py 
user = {"id": 1, "role": "root"}
print("Ваша роль:", user["role"])
> Ваша роль: root

Ошибки (Try)

Защита скрипта от падения.

safe.py 
try:
    print(10 / 0)
except:
    print("Ошибка: Деление на ноль!")
> Ошибка: Деление на ноль!

Форматирование (f-строки)

Удобный способ встраивать переменные прямо в текст. Полезно для генерации динамичных логов или сообщений.

logs.py 
unit = "Medic"
target = "Heavy"
hp = 150

print(f"[{unit}] Status: Ready. Target: {target}. HP: {hp}")
> [Medic] Status: Ready. Target: Heavy. HP: 150

Библиотеки (Import)

Python силен своими модулями. Импортируя библиотеки, мы получаем готовый крутой функционал — от математики до ИИ.

randomizer.py 
import random

seed_key = random.randint(1000, 9999)
print("Сгенерирован ключ доступа:", seed_key)
> Сгенерирован ключ доступа: 4815

Работа с файлами

Скрипты часто должны сохранять информацию. Запись логов или чтение конфигов — база для любого софта.

writer.py 
with open("system_log.txt", "w") as file:
    file.write("Инициализация системы успешна.")

print("Данные успешно записаны на диск.")
> Данные успешно записаны на диск.
