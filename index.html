Урок 1: Что такое переменные?

Представь, что переменная — это коробка, на которой ты написал маркером имя, а внутрь положил какое-то значение. В Python не нужно заранее говорить, какая это коробка, язык сам всё поймет.

Давай создадим данные для персонажа:

main.py 
# Данные игрока
character = "Heavy"
hp = 300
is_medic_ready = True

print(character, "имеет", hp, "здоровья.")
> Heavy имеет 300 здоровья.

Урок 2: Математика и автоматизация

Компьютеры созданы для вычислений. В Python можно складывать +, вычитать -, умножать * и делить /. Давай напишем скрипт, который посчитает, сколько времени займет генерация изображений нейросетью.

calc.py 
# Расчет времени генерации
images_count = 50
time_per_image = 2.5 # секунды

total_time = images_count * time_per_image
print("Для генерации 50 картинок потребуется секунд:", total_time)
> Для генерации 50 картинок потребуется секунд: 125.0

Урок 3: Ветвления логики (if/else)

Программа должна уметь принимать решения. Если условие верно (True), выполняется один блок кода, если нет (False) — другой.

logic.py 
damage_taken = 150
current_hp = 300

if damage_taken >= current_hp:
    print("Персонаж уничтожен!")
else:
    print("Медик! Срочно нужен хил!")
> Медик! Срочно нужен хил!
