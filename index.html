Урок 1: Переменные

Переменные

Переменные — это коробки для данных. Мы создаем имя и кладем туда значение.

script.py 
name = "Gamer"\nlevel = 15\nprint(f"Привет, {name}!")
> Привет, Gamer!

Математика

Python умеет считать быстрее любого калькулятора.

calc.py 
damage = 50\ncrit = 1.5\nprint("Урон:", damage * crit)
> Урон: 75.0

Условия (if)

Проверка условий позволяет программе принимать решения.

logic.py 
hp = 20\nif hp < 30:\n    print("Нужна аптечка!")
> Нужна аптечка!

Списки (Lists)

Список — это набор предметов (инвентарь) под одним именем.

items.py 
inv = ["Меч", "Щит", "Зелье"]\nprint("В руках:", inv[0])
> В руках: Меч

Циклы (for)

Циклы повторяют действие для каждого элемента в списке.

loop.py 
for x in range(3):\n    print("Выстрел!")
> Выстрел!
> Выстрел!
> Выстрел!

Функции (def)

Функция — это блок кода, который можно вызывать по имени много раз.

func.py 
def heal():\n    print("Здоровье +10")\n\nheal()
> Здоровье +10

Словари (dict)

Словари хранят данные в формате "Ключ: Значение".

data.py 
hero = {"atk": 10, "def": 5}\nprint("Атака:", hero["atk"])
> Атака: 10

Ошибки (try/except)

Позволяют программе не "вылетать", если что-то пошло не так.

safe.py 
try:\n    print(10/0)\nexcept:\n    print("Ошибка деления!")
> Ошибка деления!

F-строки

Самый быстрый и красивый способ объединять текст и переменные.

text.py 
user = "Admin"\nprint(f"Вход в систему: {user}")
> Вход в систему: Admin

Импорт (библиотеки)

Используйте чужой готовый код для своих задач (например, рандом).

random_val.py 
import random\nprint(random.randint(1, 100))
> 42 (случайно)

Файлы

Python может создавать, записывать и читать текстовые файлы.

file.py 
with open("log.txt", "w") as f:\n    f.write("Log active")\nprint("Файл сохранен")
> Файл сохранен

Классы (ООП)

Класс — это чертеж для создания сложных объектов.

oop.py 
class Bot:\n    name = "Imba"\n\nb1 = Bot()\nprint(b1.name)
> Imba
