Уровень 1: Подготовка окружения и первая команда

Добро пожаловать в PyAcademy, рекрут. Ты сделал первый шаг к тому, чтобы стать архитектором цифровых миров. Python — это не просто язык программирования, это твой универсальный инструмент, твой световой меч в мире данных.

На этом уровне мы не будем устанавливать сложные IDE. Наша цель — понять базовый протокол общения с компьютером. Самая главная команда в арсенале любого хакера — это print(). Она заставляет машину вывести данные на экран.

  • Команда print — это функция вывода.
  • Текст внутри скобок должен быть заключен в кавычки (одинарные или двойные).

Ячейки Памяти: Переменные

Уровень 2: Хранение и манипуляция данными

Код не может существовать без данных. Чтобы программа помнила информацию, нам нужны **переменные**. Представь их как подписанные коробки в огромном хранилище оперативной памяти.

В Python всё просто: ты придумываешь имя (например, user_name), ставишь знак равно = (это оператор присваивания), и кладешь туда значение.

В коде ниже мы создаем три ячейки: для текста, целого числа и числа с плавающей запятой.

📄 variables.py
# Записываем данные агента
agent_name = "Neo"          # Строка (String)
access_level = 10           # Целое число (Integer)
integrity = 0.98            # Число с запятой (Float)

print("Агент:", agent_name)
print("Уровень допуска:", access_level)
Output
>>> Агент: Neo
>>> Уровень допуска: 10
